# AI Video Dubber SaaS (Scale Plan Edition)

A professional-grade AI video dubbing platform built for hackathons and scale. This project automatically transcribes, translates, clones your voice, and generates dubbed videos in multiple languages with perfect synchronization.

## 🚀 Vision
Empower creators to reach a global audience with a single click. No more language barriers.

## ✨ Core Features
- **7-Step AI Pipeline**: Full automation from upload to final export.
- **Instant Voice Cloning**: Powered by ElevenLabs Scale Plan.
- **High-End UI**: Premium dashboard with glassmorphism and real-time tracking.
- **Multilingual Support**: Generate `video_en.mp4`, `video_es.mp4`, `video_hi.mp4`, and more.
- **Podcast Mode**: Full support for audio-only dubbing.
- **Automatic Subtitles**: SRT generation for every project.
- **Cost Efficient**: Optimized to use free-tier services where possible.

## 🛠️ Tech Stack
- **Frontend**: React + Vite + Framer Motion (Glassmorphism UI)
- **Backend**: Node.js + Express + FFmpeg
- **Database**: Firebase Firestore
- **Storage**: Firebase Cloud Storage
- **AI Engines**: ElevenLabs (Native Video Dubbing, Transcription, Translation, Voice Cloning)
